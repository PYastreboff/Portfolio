How To Use

To Use this website, scroll down just as you would any other webpage. The only difference is this website uses parralax illusions to create the sense that objects are moving around the screen, when in reality you are scrolling down the page.


Other animations included in this website include scroll animations in which the content will fade in from the sides of the screen. These animations do not require you to scroll, they run automatically.


⚠️ Warning: This parralax scroll illusion along with all other animations on this page will cause lag, These website has been tested to run smoothly on most machines, however if it doesn't run smoothly on your computer consider scrolling through the page for the animations to load.


If any further problems arise, please feel free to contact me. My Socials are provided at the bottom of the website.


Enjoy!
